# Resume-Builder

Welcome to my first official android app Resume Builder, a simple way to generate a PDF résumé / CV (curriculum vitæ).Feel free to review the source code, fork the repository, and send pull requests


## Functionalities
------------
- User can fill up all the field with personal data
- After completing that, he/she can save that in a pre defined PDF format
